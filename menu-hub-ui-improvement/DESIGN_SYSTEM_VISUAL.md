## ⚡ 动画系统详细展示

### 过渡时间

```
快速反馈（200ms）
点击按钮 → 立即响应
用途：按钮、输入框、小元素

标准过渡（300ms）
页面切换 → 平滑过渡
用途：页面、模态框、卡片

缓慢过渡（400ms）
重要动画 → 引起注意
用途：关键动画、提示
```

### 缓动函数

```
平滑动画（cubic-bezier(0.4, 0, 0.2, 1)）
╱─────────────────────────────────╲
│ 开始快，结束慢，看起来很自然     │
│ 用途：大多数动画                │
╲─────────────────────────────────╱

弹性动画（cubic-bezier(0.68, -0.55, 0.265, 1.55)）
╱─────────╲
│ 有弹性感 │╱─────╲
│ 用途：   │      ╲
│ 强调动画 │       ╲
╲─────────╱        ╲
                    ╲

线性动画（linear）
─────────────────────────────────
│ 匀速运动，看起来机械感
│ 用途：加载、旋转
```
